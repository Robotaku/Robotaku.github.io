## ✨ 主な機能（モジュール）

- **📊 総合ダッシュボード**: 習得単語数や各分野の進捗から「推定スコア」を算出・表示します。
- **📘 単語・語彙学習**: 分野別のフラッシュカード機能（覚えた/わからない）と、4択の確認テスト。
- **🧩 文法問題**: 品詞や時制などの頻出問題を、1問ずつの解説付きで演習。
- **🎧 リスニング**: ブラウザの音声合成機能を利用した英語読み上げ問題（再生速度の調整可能）。
- **📰 長文読解**: ビジネス文書の読解問題。日本語訳の表示/非表示を切り替え可能。
- **☁️ クラウド同期**: Google Driveの `appDataFolder`（アプリ専用の隠しフォルダ）を利用し、進行状況を自動でセーブします。
